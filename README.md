# Loan Approval Analysis using SQL

## Project Overview
This project analyzes loan application data to identify factors influencing loan approval decisions using SQL.

## Tools Used
- PostgreSQL
- pgAdmin

## Business Questions Answered
- How does a credit score impact loan approval?
- Is income higher for approved applicants?
- Do graduates have higher approval rates?
- Does the loan amount affect rejection?
- How do assets influence approval decisions?

## Project Structure
- `01_data_overview.sql` → Dataset understanding & exploration
- `02_business_analysis.sql` → Business-driven SQL analysis

## Skills Demonstrated
- Data exploration
- CASE statements
- Aggregations & grouping
- Business-oriented thinking
- Clean, readable SQL
